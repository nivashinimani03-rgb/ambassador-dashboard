🌟 Hushh Ambassify Dashboard

📌 Overview

Hushh Ambassify is a web-based dashboard designed to track and analyze the performance of campus ambassadors. It provides insights into signups, engagement, and growth using interactive charts and reports.

---

🚀 Features

- 🔐 Login system for secure access
- 📊 KPI metrics (Signups, Engagement, Conversion Rate)
- 📈 Weekly growth analysis
- 🏆 Leaderboard for top performers
- 📉 Interactive charts using Chart.js
- 🧑‍🤝‍🧑 Ambassador directory with performance filters
- 📂 Upload CSV/JSON data
- 🤖 AI chatbot for quick insights

---

🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Chart.js
- Font Awesome

---

📷 How It Works

1. Login to access the dashboard
2. View performance metrics and charts
3. Analyze weekly growth
4. Track top ambassadors
5. Upload new data for updates
